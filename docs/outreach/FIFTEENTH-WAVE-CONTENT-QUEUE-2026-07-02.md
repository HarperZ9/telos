## Post 87

The formal replay ladder moved from a two-face cancellation lemma to a finite cyclic-sum theorem.

`pathDiffSum start xs + (start - lastValue start xs) = 0`

`sum_i (a[i+1] - a[i]) = 0`

Still not the continuous PDE theorem. But it is a real rung.
