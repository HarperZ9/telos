## Abstract

Project Telos is building proof-centered research packets that preserve the
difference between source intake, architectural inference, executable evidence,
verification verdicts, and public claims. This official local copy records a
biology/network-intelligence packet for fungal, mycorrhizal, and plant signaling
literature. The verified artifact is a ten-item Gather corpus with ten distinct
bodies and a matching run digest. The architecture result is a bounded
hypothesis: a hyphal context protocol for sparse signal routing and
receipt-triggered evidence retrieval.
