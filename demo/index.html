Telos Studio effects

Your browser can still use the text controls and scene receipt if canvas rendering is unavailable.

Creative engine

The whole engine, not only the surface.

Telos now treats visual effects, sound, typography, math, node graphs, and verification as one creation system. The live Studio is the front panel; the engine map below is the contract for reviving old organs and plugging them into CLI, MCP, IDE, TUI, and application hosts.

Dither, pixel sorting, halftone, CRT, chromatic split, poster grids, glitch, and export-ready media passes.

Sound and media

Math and physics

N-dimensional geometry, flow fields, harmonographs, particle fields, clustered light bins, Gaussian splat fields, and measurable overlays for mathematicians, physicists, and educators.

Certificate loop

The verifier has to be able to refuse.

The proof renders a four-dimensional cube, perceives it two ways, checks the recovered invariant against a criterion the model did not author, then shows both a certified pass and an honest `UNVERIFIABLE`.
